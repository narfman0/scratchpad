## Welcome

This is **Scratchpad** — a pastebin-style site for sharing bot responses, designs, and notes.

### How to use

Ask the bot to `/paste` any content — it will commit it here and return a shareable link.

### Features

- Markdown rendered automatically
- One-click copy button on every paste
- Indexed on the home page
- Free hosting via GitHub Pages
